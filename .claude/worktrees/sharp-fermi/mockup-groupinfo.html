What changes in each

All options: back button → arrow only  ·  group name Inter 700 → Cormorant Light 36px  ·  leave group text gets lighter weight
A — Full Option D
Tracked uppercase labels throughout

Adds "MEMBERS" section header above the list. "Created by" and section label both use the Inter-Regular 10px tracked style used on field rows and CollapsibleSection headers. Most editorial.

Inter 700 26px → Cormorant Light 36px
Inter 14px → Inter 10px tracked
B — Serif + natural sub
Cormorant title, natural-weight sub

Serif title at 36px, but "Created by Chloe" stays as a readable 13px sentence (not uppercase/tracked). Member names move to Inter 500 — slightly lighter than bold but still clear. Softer feel.

Inter 700 26px → Cormorant Light 36px
Inter 600 member names → Inter 500
C — Compact sub-label
Count + creator on one line

Mirrors the group chat header sub-label pattern: "3 members · created by Chloe" as a single small tracked line. Most space-efficient, removes the separate "Created by" text, consistent with chat header.

Inter 700 26px → Cormorant Light 36px
separate created-by line → 3 members · created by X
